# LOFT Liquidity Providing Program

# Quickstart

## Installation

```
npm install
```

## Run tests

```
npm test
```

## deplpy

```
yarn hardhat deploy --network bsctestnet  --tags Dev
```

## verify

```
yarn hardhat --network bsctestnet  etherscan-verify
```

## export

```
yarn hardhat export  --network bsctestnet --export  ./export/lp
```
